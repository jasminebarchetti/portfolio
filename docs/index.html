Assignment #2 — User Feedback

Feedback Log (Anonymized)

User Role (Generic) Key Finding (Paraphrased) HCI Theme Action Taken (Decision)
User A Recruiter I cannot see the 'View Work' button. It completely blends in, making it unclear what the next step is. Visibility / Affordance Make the CTA button solid and high-contrast.
User B Marketing Manager The large name doesn't state your job. I need the professional role visible right away. Clarity / Mapping Add a professional sub-headline (e.g., Student Business Informatics).
User C Collaborator The scrolling Ticker and the CTA cancel each other out due to low contrast. The Ticker itself is too faint to read. Aesthetics / Cognitive Load Increase the Ticker contrast (dark background/light text).
User D UX Designer The style is elegant, but there is inconsistency; interactive elements (buttons/links) use different font styles. Consistency / Standards Standardize the fonts and colors for all interactive elements.

Implemented Changes (Before/After)

1. CTA Visibility (P1)

Improved Affordance by making the primary action solid, changing it from a low-contrast outline to a dark fill.

2. Title Clarity (P2)

Improved Mapping by adding a professional sub-headline to immediately communicate the student's role.
